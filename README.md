#Description
Use with qmk. Map apple USB keybaord A1242 to windows 10. It was tested for converter/usb_usb/hasu.

###Mappings:
- CMD + a -> CTRL + a
- CMD + f -> CTRL + f
- CMD + m -> CTRL + m (minimize current window
- CMD + t -> CTRL + t 
- CMD + q -> ALT+ F4 
- CMD + z -> CTRL + z
- CMD + LEFT -> HOME
- CMD + END -> END
